# Identity and Access Management – PowerShell Scripts

Este repositório contém scripts PowerShell voltados para automação,
segurança e governança de identidades em ambientes Microsoft.

## Tecnologias
- Active Directory On-Premises
- Microsoft Entra ID (Azure AD)
- Exchange On-Premises e Online
- PowerShell
- Microsoft Graph

## Casos de uso
- Automação de ciclo de vida de usuários
- Auditoria de acessos e contas privilegiadas
- Implementação e verificação de MFA
- Governança de grupos e permissões
- Apoio a controles de segurança e conformidade (ISO 27001 / LGPD)

## Objetivo
Reduzir riscos de acesso indevido, aumentar eficiência operacional
e fornecer evidências técnicas para auditorias de segurança.
